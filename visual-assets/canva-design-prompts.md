# Canva Design Prompts

Use these prompts to create branded visuals for this eligibility verification portfolio project.

## Brand Colors

- Primary Background: White `#FFFFFF`
- Accent Color: Tennessee Orange `#FF8200`
- Font Color: Black `#000000`

---

## Visual 1 — Eligibility Verification Workflow Map

Create a clean healthcare operations workflow diagram on a white `#FFFFFF` background. Use black `#000000` text and Tennessee Orange `#FF8200` as the accent color. Design a vertical process map with orange circular nodes connected by thin black or warm gray lines. Include these workflow steps: Patient Scheduled, Insurance Information Collected, Coverage Status Verified, Member ID Confirmed, Payer and Plan Type Validated, Coordination of Benefits Reviewed, Authorization Requirement Checked, Patient Responsibility Communicated, Account Cleared Before Service. Use large whitespace, clean sans-serif typography, and a professional healthcare analyst style. Add a small footer that says: Created by Kori Pickle.

---

## Visual 2 — Front-End Denial Prevention Diagram

Create a professional white-background infographic showing how front-end eligibility verification prevents downstream denials. Use black text and Tennessee Orange accents. On the left, show upstream checkpoints: Coverage Active, Correct Member ID, Correct Payer, COB Reviewed, Authorization Checked, Patient Responsibility Communicated. On the right, show downstream risks prevented: Claim Denials, Rejections, A/R Delays, Billing Confusion, Rework, Patient Frustration. Connect the two sides with orange arrows. Keep the layout clean, balanced, and recruiter-readable. Add footer: Created by Kori Pickle.

---

## Visual 3 — Eligibility Failure Points Summary

Design a clean portfolio graphic using a white background, black text, and Tennessee Orange section dividers. Title: Where Eligibility Verification Breaks. Create six cards labeled Inactive Coverage, Incorrect Member ID, Wrong Payer Selected, COB Not Reviewed, Authorization Requirement Missed, Patient Responsibility Not Communicated. Each card should include a short downstream impact. Use orange icons or small orange node markers. Keep the visual minimal, premium, and healthcare operations focused. Add footer: Created by Kori Pickle.

---

## Visual 4 — Patient Access Checkpoint Graphic

Create a clean patient access checkpoint diagram with a white background, black typography, and Tennessee Orange accent lines. Show a checklist-style layout with these checkpoints: Insurance Active, Demographics Matched, Member ID Confirmed, Payer Rules Reviewed, COB Verified, Authorization Requirement Checked, Financial Responsibility Communicated. Use orange check icons and clean spacing. Add footer: Created by Kori Pickle.

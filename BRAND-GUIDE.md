# Brand Guide

## Primary Brand Colors

| Brand Role | Color | Hex Code | Use |
|---|---|---:|---|
| Primary Background | White | `#FFFFFF` | Main background, documentation space, portfolio visuals, case study layouts |
| Accent Color | Tennessee Orange | `#FF8200` | Headings, dividers, workflow nodes, callout boxes, key metrics, emphasis labels |
| Font Color | Black | `#000000` | Main text, tables, labels, process steps, section titles |

## Visual Identity

This project uses a clean healthcare operations style built around clarity, structure, and analyst-level communication.

## Design Rules

- Use white as the dominant background.
- Use black for primary typography.
- Use Tennessee Orange for emphasis only.
- Keep spacing clean and professional.
- Use tables, workflows, and process maps instead of decorative clutter.
- Make every visual explain a revenue cycle problem or prevention step.

## Suggested Canva Layout

- White background: `#FFFFFF`
- Black headline text: `#000000`
- Tennessee Orange accent line: `#FF8200`
- Orange circular workflow nodes
- Thin black or warm gray connector lines
- Large whitespace around each section
- Footer text: `Created by Kori Pickle`

## Visual Tone

Professional, clean, operational, healthcare-focused, and recruiter-readable.
